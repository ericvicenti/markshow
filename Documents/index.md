![Introducing Markshow](/.assets/img/markshow_banner.png)

MarkShow is a node.js server which displays a folder of markdown as a documentation website.

## Installation

MarkShow is tested on node v.8.x

1. Download the repo
2. `npm install`
3. `node server`
